## [0.0.86] - 2025-10-20

### Added
- New `rumdl.format.autoFixOnSave` configuration option to automatically fix markdown issues when saving files
- Proper TOML parser (`@iarna/toml`) for robust configuration validation
- Integration tests for config validation and pyproject.toml extraction
- Two-phase validation system (TOML syntax validation → semantic validation)
